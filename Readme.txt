ThreadBeans(PC side) is a Desktop application developed in java for the PC side for the android application ThreadBeans.
It is a netbeans project folder.
The src in ThreadBeans(PC side) contains the source for the PC side of application.
Run the MainClass.class to start application.


Another folder contains the source for the android application, named
ThreadBeans(Android side).
Import it in android studio.


The application is partially developed. But still the basic working is implemented.
